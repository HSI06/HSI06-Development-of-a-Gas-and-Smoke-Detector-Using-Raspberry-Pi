#  Development of a Gas and Smoke Detector Using Raspberry Pi

라즈베리파이와 MQ-2(FC-22) 센서를 활용하여  
실시간으로 가연성 가스 및 연기를 감지하고,  
위험 상황 시 능동 부저를 통해 경고를 알리는 **AIoT 방재 시스템 프로젝트**입니다.

---

##  Project Overview

본 프로젝트는 반도체식 가스 센서의 저항 변화 원리를 이해하고,  
라즈베리파이 GPIO를 통해 수집된 디지털 신호를 소프트웨어로 제어하는 것을 목적으로 합니다.

### 🔹 주요 기능
- 가스 및 연기 실시간 모니터링  
- 위험 감지 시 능동 부저를 통한 경고음 발생 
- 하드웨어 가변저항을 통한 감지 민감도 최적화  

---

##  작동 원리
가스/연기 감지 센서(MQ-2)의 원리본 프로젝트의 핵심은 MQ-2 센서 내부에 있는 **산화물 반도체($SnO_2$)**의 전기적 성질 변화를 이용하는 것입니다.
- 산소 흡착 (평상시): 깨끗한 공기 중에서 센서 표면의 산화주석($SnO_2$)은 산소 이온을 흡착합니다. 이 산소 이온들이 전자 흐름을 방해하여 센서의 전기 저항이 매우 높은 상태를 유지합니다.
- 환원 반응 (감지 시): 가연성 가스(LPG, 부탄 등)나 연기 입자가 센서에 닿으면, 표면에 붙어있던 산소 이온과 반응하여 이를 떼어내는 환원 반응이 일어납니다.
- 저항 감소 및 신호 발생: 산소가 떨어져 나가면서 방해받던 전자들이 자유롭게 흐르게 되고, 결과적으로 소자의 저항이 급격히 감소합니다.
- 디지털 출력: 센서 모듈은 이 저항 변화를 전압 신호로 바꾸어, 설정된 임계값보다 낮아질 경우 Low(0) 신호를 라즈베리파이 GPIO 17번 핀으로 전송합니다.

##  Hardware Configuration

실험에 사용된 하드웨어 구성 사양입니다.

| 부품명 | 연결 정보 (Pin) | 역할 |
|--------|----------------|------|
| MQ-2 Sensor | GPIO 17 (Input) | 가스/연기 감지 및 디지털 신호 출력 |
| Active Buzzer | GPIO 18 (output) | 가스/연기 감지 시 경고음 발생 |
| Raspberry Pi 5 | - | 메인 컨트롤러 및 데이터 처리 |
| Jumper Wire | Female-to-Male | 센서와 파이 간 직접 연결 |

##  Software & Logic

- **Language:** Python 3.x  
- **Library:** RPi.GPIO 또는 gpiozero  

### 🔹 핵심 로직

1. **Initialization**  
   - GPIO 핀 설정 및 센서 예열 단계 수행  

2. **Monitoring**  
   - 무한 루프를 통해 센서의 디지털 출력값 상시 감시  

3. **Action**  
   - 가스 감지 (Low 신호 발생) 시 능동 부저(Active Buzzer)에 신호를 출력하여 경고음 발생

- **감지 성능:**
